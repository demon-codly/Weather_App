🌤️ Weather App
A simple and responsive web-based weather application that allows users to search for and view real-time weather information for any location worldwide.

🔍 Features
🔄 Live weather updates using WeatherAPI

🌡️ Displays current temperature in Celsius

🕒 Shows local date and time for searched location

🌍 Location-based weather condition (like Clear, Mist, Rain, etc.)

📱 Responsive UI with clean and minimal design

🔎 Search bar to check weather for any city

🚀 Technologies Used
HTML5 & CSS3 for structure and styling

JavaScript (ES6+) for dynamic data handling and API integration

WeatherAPI for fetching real-time weather data

🧠 How It Works
User enters a city name in the search bar.

The app fetches live weather data using WeatherAPI.

It updates the UI to display temperature, weather condition, and local time.

📦 Setup
To run locally:

Clone the repo or download ZIP

Open index.html in any modern browser

Enjoy live weather updates!

Note: You need a valid WeatherAPI key to make API requests.
